A BRIEF OVERVIEW OF NOTES APP

In this present digital world, many are shifting their work from manual to digital way.
In order to contribute a little to this transformation, NOTES APP plays a very effective role.
It is exciting to wonder how organized it would look if people manage their A to Z digitally by making notes!!
This project deals with such interesting things.
The project’s features are as follows:
• It includes Sign-in or sign-up landing page for user authentication
• The user can view all the notes written previously,
add new notes to the database also edit them when necessary
• User can see the dates and timings of the creation of the note
• Very user friendly and easy to access.
